# 🎬 소울싱크(Soul Sync) 영상 캠페인 스토리보드 기획안

## 1. 브랜드 아이덴티티
* **브랜드명:** 소울싱크 (Soul Sync)
* **정체성:** "0.001%의 오차도 없는 완벽한 판타지 세계와의 감각 동기화" (캡슐을 통한 이세계 전이)

## 2. 캠페인 목표 및 핵심 메시지
* **캠페인 목표:** 캡슐에 접속하는 순간 펼쳐지는 압도적인 판타지 세계를 시각화하여 유저의 호기심 자극
* **핵심 메시지:** "당신의 영혼이 깨어나는 곳, 소울 싱크."

## 3. 사용 도구 목록 (Tech Stack)
* **Gemini 3.5 Pro:** 스토리보드 씬(Scene)별 세부 기획, 이미지/영상 생성을 위한 프롬프트 엔지니어링 및 감성적인 내레이션 텍스트 작성
* **OpenAI Sora 2:** 텍스트 프롬프트를 기반으로 씬별 고해상도 초실사 비디오(동영상) 일괄 생성
* **CapCut:** BGM 추가 및 최종 영상 컷편집
* **Clyde:** 각 Scene에 맞는 내레이션 음성 생성
